Leetcode 20: Valid Parentheses

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the
input string is valid. An input string is valid if:
1.Open brackets must be closed by the same type of brackets.
2.Open brackets must be closed in the correct order.
3.Every close bracket has a corresponding open bracket of the same type.

One-pass without special data structure

The key here is to guarantee each pair of parentheses are matched. A one pass scan can keep track of the counts of open parentheses, but cannot keep track of the ordering of different types of them.

One-pass with Stack

A stack can be introduced to keep track of the location sequence of 3 types of parentheses. The
paratheses matching is last in first out. Which matches perfectly with the stack.
Time complexity: O(N)
Space complexity: O(N)

Code 


class Solution {
    public boolean isValid(String s) {
        Stack stack = new Stack<>();
        for (char c : s.toCharArray()) {
            switch(c) {
                case '(':
                case '[':
                case '{':
                    stack.push(c);
                    break;
                case ')':
                    if (stack.isEmpty() || stack.peek() != '(')
                        return false;
                    stack.pop();
                    break;
                case ']':
                    if (stack.isEmpty() || stack.peek() != '[')
                        return false;
                    stack.pop();
                    break;
                case '}':
                    if (stack.isEmpty() || stack.peek() != '{')
                        return false;
                    stack.pop();
                    break;
            }
        }
        return stack.isEmpty();
    }
}
